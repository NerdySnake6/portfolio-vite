# Портфолио на Vite + Bootstrap

## Быстрый старт

```bash
# 1. Клонировать репозиторий
git clone git@github.com:NerdySnake6/portfolio-vite.git
cd portfolio-vite

# 2. Установить зависимости
npm install

# 3. Запустить в режиме разработки
npm run dev
Сборка для продакшена
npm run build
Готовый сайт будет в папке dist/

Технологии
Vite

Bootstrap 5

SCSS

Структура проекта
portfolio-vite/
├── index.html
├── package.json
├── src/
│   ├── main.js
│   └── scss/
│       └── custom.scss
└── public/

Команды
Команда	Описание
npm run dev	Запуск dev-сервера
npm run build	Сборка проекта
npm run preview	Предпросмотр собранного проекта
