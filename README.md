# Портфолио на Vite + Bootstrap

## Быстрый старт

```bash
# 1. Клонировать репозиторий
git clone git@github.com:NerdySnake6/portfolio-vite.git
cd portfolio-vite

# 2. Установить зависимости
npm install

# 3. Запустить в режиме разработки
npm run dev
```

## Сборка для продакшена

```bash
npm run build
```

Готовый сайт будет в папке `dist/`

## Технологии

- [Vite](https://vitejs.dev/)
- [Bootstrap 5](https://getbootstrap.com/)
- [SCSS](https://sass-lang.com/)

## Структура проекта

```
portfolio-vite/
├── index.html
├── package.json
├── src/
│   ├── main.js
│   └── scss/
│       └── custom.scss
└── public/
```

## Команды

| Команда | Описание |
